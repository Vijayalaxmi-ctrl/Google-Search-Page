Google Homepage Clone
This project is a simple Google Homepage UI Clone created using HTML and CSS.  
It is built for learning and practicing basic frontend development concepts.

Project Description
The goal of this project is to recreate the visual layout of Google’s search homepage, including:
- Google logo
- Search input box
- Google Search and I'm Feeling Lucky buttons
This project focuses only on UI design, not real search functionality.

Technologies Used
HTML5 – Structure of the webpage  
CSS3 – Styling and layout
